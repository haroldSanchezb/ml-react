# ml-react

## Install

``` bash
npm install
```

## Get Schema

```
npm run get-schema
```

## Get Schema Files

```bash
npm run relay
```

## Start Local Server

*currenlty development mode*

```
npm start
```

# Updating Schema GraphQL

In a tab:

```
npm start
```

then, run in another tab:

```
npm run get-schema
```
